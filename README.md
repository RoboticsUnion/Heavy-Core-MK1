# Heavy-Core-MK1
Heavy-Core-MK1 is the development of a modular robotic arm platform.
The goal is to create a scalable, affordable robotics ecosystem where hardware and software are engineered together from the ground up.

The first two axes have already been designed, printed, assembled, and successfully tested. In parallel,
we are developing our own control software for precise motion control.

# Our Vision:

We aim to build a 6 DOF robitic arm that:
  - Is upgradeable with interchangeable end-effectors
  - Is structurally optimized for strength-to-weight efficiency
  - Includes a custom-developed control software
  - Allows future automation features
  - Serves as a platform for robotics development

# Custom Control Software

Our robotic arm will not rely solely on position control.
We are developing a control application that will include: 
  - Real-Time position feedback
  - Inverse kinematics calculations
  - Intuitive controls
  - Integrated arm simulation to prevent collisions or overload situations
  - Expandable architecture for future features

# Preliminary Technical Specifications
- Target DOF: 6 DOF
- currently functional: 2 DOF
- Actuators: Stepper motors
- Structure: Aluminum & PLA (Stronger 3D-printed material planned)
- Control system: Arduino & ESP32 based
- Sofware: Custom standalone desktop or mobile application

# Why this project stands out

Heavy-Core is more than a 3D-printed robotic arm.
It is the foundation for a modular robotics ecosystem that is mostly custom designed.

Mechanical engineering, electronics and software development are 
combined in one unified system, driven by the abition and passion of young engineers.
